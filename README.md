# Offline Toolbox Android

Offline-first production-ready Android starter for an all-in-one Productivity, Finance, and Developer Toolkit.

## Scope

No backend. No login. No ads. No payments. All processing happens locally on device.

## Stack

- Kotlin
- Jetpack Compose + Material 3
- MVVM + StateFlow
- Navigation Compose
- Room for local history/counter sessions
- DataStore for settings, favorites, recent tools, timer settings
- Hilt for DI
- Kotlinx Serialization

## Features scaffolded

### Productivity
- Pomodoro Timer
- Stopwatch & Timer
- Breathing Pacer
- Tally Counter

### Finance
- EMI Calculator
- Split Bill Calculator
- Discount Calculator
- Rule of 72 Calculator

### Developer
- JSON Formatter
- URL Encoder/Decoder
- Hash Generator
- Lorem Ipsum Generator
- Color Converter

## Import

Open this folder in Android Studio and sync Gradle.

Package name: `com.puneeth450.offlinetoolbox.app`
